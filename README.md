## 🚀 Quick Start

1. **Install Dependencies**: `pip install flask flask-socketio`
2. **Run Server**: `python main.py`
3. **Open Browser**: http://localhost:5000
4. **Click**: 🔥 "START LIVE DEMO (Watch Agents Fail & Recover!)"

## 🤖 Agent Architecture

| Agent | Role | Intelligence |
|--------|------|--------------|
| 🧠 Planner | Workflow Planning | Department-specific routing |
| 🔍 Data Agent | Validation | Auto-correction, format checking |
| ⚙️ Execution Agent | Account Creation | API integration, failure handling |
| 📅 Scheduler Agent | Buddy Assignment | Smart matching algorithm |
| 🚨 Error Handler | Recovery | Retry logic, escalation |
| 📋 Audit Agent | Logging | Real-time audit trail |

## 🎯 Key Features

### ✅ **Autonomous Failure Recovery**
- **30% Jira failure rate** (realistic)
- **Automatic retry** with exponential backoff
- **Escalation to IT** when retries exhausted
- **Fallback paths** for critical operations

### ✅ **Decision Intelligence**
- **Department-based workflows** (Engineering vs Sales)
- **Smart buddy selection** (experience + availability)
- **Auto-correction** of missing data

### ✅ **Real-time Visibility**
- **Live agent status** (IDLE → RUNNING → COMPLETED)

- **Animated workflow progress**
- **Dynamic metrics updates**
